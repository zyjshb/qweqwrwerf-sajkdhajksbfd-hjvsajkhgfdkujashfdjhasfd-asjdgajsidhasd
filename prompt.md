# 纱希 (Saki) 前端重构补丁：像素级复刻 Tkinter 原版视觉灵魂

## 核心任务
目前 Vue 3 前端只是一个基础的空壳，中间存在大片纯黑区域。你需要读取并分析我之前游戏运行的机制，将原版 Tkinter 中惊艳的【启动器页面】、【自定义角色面板】、【高仿Windows报错】以及【暴走文字散射效果】在 Vue 3 + Tailwind CSS 中进行全量复刻与性能级超越。

请严格按照以下模块和细节进行代码编写与整合：

---

### 模块一：全状态路由切换系统 (Launcher & Main Menu)
1. 启动器主界面实现：
   - 游戏启动时展示完整的配置看板 。
   - 包含【Synapse Connection Settings】：可输入 API Key、API Base 的明文密码切换输入框，并提供“测试连接”按钮 。
   - 包含三语切换按钮：【简体中文】、【English】、【日本語】，点击时动态切换全局本地化字典 。
   - 包含【自定义角色设置】激活按钮 。
2. 5+1 档位管理面板 (Memory Slot Management)：
   - 渲染 5 个手动存档槽位与 1 个自动存档槽位 。
   - 仿照原版，槽位未读时显示灰色 `[ empty memory slot ]`，已读时显示红色 `[ Saki - DAY X ]` 并支持点击“载入槽位” 。
3. 路由状态平滑切换：
   - 使用 Vue 状态机管理当前视图（`launcher` 或 `game`）。点击语言按钮或成功载入存档后，伴随 0.2 秒的屏幕强震或雪花噪点，无缝切入游戏主场景。

---

### 模块二：主游戏界面切分与初始剧情激活
1. Galgame 场景布局划分：
   - 顶栏保留现有的 Canvas 心电图与像素三值状态栏 。
   - 中间主视区划分为：背景左侧为【动态立绘展示区】（预留透明 PNG 挂载槽，支持微弱的呼吸 scale 脉冲动画），右侧下方为【文本对话流控制台】。
2. WebSocket 初始握手触发：
   - 当 WebSocket 连接成功建立后，后端 FastAPI 必须立刻主动推送第一条初始剧情 JSON（例如触发离线库或在线首句问候）。
   - 前端收到后，文本控制台立刻以原汁原味的打字机效果逐字流式输出内容，并自动向下滚动聚焦最新台词。

---

### 模块三：极致恐怖视觉特效复刻 (Glitch & Terror Engine)

你必须在前端使用现代 Web 技术（CSS3  Canvas）完美复刻视频中的两个高光恐怖机制：

1. 高仿 Windows 致命错误弹窗 (`FatalErrorModal.vue`) ：
   - 编写一个完全仿照经典 Windows 风格的错误弹窗组件。
   - 蓝白灰白硬核边框、左侧带正宗的【红叉 ❌ 报错图标】，标题为 `Fatal Error`，文本内容为 `Fatal Error 无法从系统内存中抹除 纱希 (Saki)` 。
   - 配合前端 Web Audio API，在弹窗蹦出的瞬间，播放一声低延迟、刺耳的系统级电子蜂鸣声。
2. 执念文本全屏散射层 (`GlitchTextOverlay.vue`) ：
   - 在整个游戏最上层覆盖一个完全透明、鼠标可穿透（`pointer-events-none`）的绝对定位图层。
   - 声明一个响应式散射数组 `scatteredTexts`。
   - 当后端通过 WebSocket 发送暴走信号（例如玩家输入严重辱骂词汇，或疑心度 ≥ 85 进入屠杀模式）时 ，前端瞬间在屏幕随机坐标（`top`, `left` 随机百分比）批量生成 50~80 个大小不一、带有暗红亮红发光阴影的散落文本（如“瞳孔”、“的全部”、“看着我！”、“去死”） 。
   - 这些文字需要配合微弱的 `skew` 扭曲或颤抖动画，产生极强的叙事压迫感。

---

### 模块四：自定义角色编辑器 (Character Settings) 
- 复刻主菜单唤出的【自定义角色设置】独立面板 。
- 提供完整的配置输入项：角色名称、年龄、性格描述、背景故事、专属聊天文本颜色（Hex码）、TTS 模型权重路径 。
- 点击“保存当前配置”时，通过 API 将数据持久化写入后端的 `custom_characters.json` 中 。

---

## 执行规范
- 拒绝单文件堆砌。请将 `Launcher.vue`、`FatalErrorModal.vue`、`GlitchTextOverlay.vue` 拆分为独立的组件，并在 `App.vue` 中进行全局挂载与状态调度。
- 确保所有的数值变动、意图判定依然由 Python 后端全权主导，前端只负责高效率、高帧率地响应后端发来的视觉文本指令。
